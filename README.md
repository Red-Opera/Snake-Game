## 주요 기능

<ul>
    <li>
        <h3>Game Rule</h3>
           - 벽 또는 자신이 꼬리에 닿으면 게임 종료 <br>
          - 꼬리 길이가 75이상이고 100초이상 버티게 되면 승리<br>
          - 게임 진행함에 따른 게임 속도 증가<br>
    </li> 
    <li>
        <h3>Function key</h3>
        위로 이동 : W, 아래로 이동 : A, 왼쪽으로 이동 : S, 오른쪽으로 이동 : D
    </li>
</ul>
